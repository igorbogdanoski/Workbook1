# MathWorksheets MK (Документација)

Ова е интерактивна веб апликација за математички работни листови, изградена со React и TypeScript.

## Структура на проектот

Апликацијата е дизајнирана да биде модуларна и лесно проширлива. Главните податоци не се "хард-кодирани" во компонентите, туку се наоѓаат во конфигурациски фајлови.

### Клучни фајлови

*   `constants.ts`: Ова е **најважниот фајл** за додавање содржина. Тука се дефинирани сите работни листови (worksheets), задачи, клучни зборови и точни одговори.
*   `types.ts`: Ги дефинира TypeScript типовите (Task, Worksheet, Keyword) кои ја гарантираат структурата на податоците.
*   `components/TaskCard.tsx`: Компонента која знае како да прикаже една задача (без разлика дали е текст, под-задачи, или стил „тетратка“).
*   `pages/WorksheetView.tsx`: Ја прикажува страницата за одреден работен лист, вклучувајќи ги табовите (Фокус, Вежбање, Предизвик) и клучните зборови.

## Како да додадете нов работен лист

Не е потребно менување на HTML или React кодот за додавање нови задачи. Следете ги овие чекори:

1.  Отворете го фајлот `constants.ts`.
2.  Најдете ја листата `WORKSHEETS`.
3.  Додадете нов објект во таа листа. Можете да го ископирате постоечкиот објект и да ги смените податоците.

Пример структура:

```typescript
{
  id: "ws-7-2", // Единствено ID за новиот лист
  title: "Собирање на дропки",
  section: "7.2 Adding fractions",
  keywords: [
    { text: "improper fraction (неправилна дропка)", relatedTaskIds: ["Q1"] }
  ],
  tasks: [
    {
      id: "Q1",
      category: "Focus", // Focus, Practice, или Challenge
      instruction_mk: "Пресметај...",
      prompt_en: "Calculate...",
      math_elements_latex: ["\\frac{1}{2} + \\frac{1}{4}"],
      answers: {
        "main": ["3/4", "0.75"] // Точни одговори (поддржува повеќе варијанти)
      }
    }
  ]
}
```

## Зачувување на податоци

*   **Точни одговори:** Се дефинираат во `constants.ts` во полето `answers`.
*   **Одговори на корисникот:** Се чуваат автоматски во прелистувачот на корисникот (Local Storage). Ако корисникот ја освежи страницата, одговорите ќе останат.

## Специфични функционалности

*   **Notebook Style:** Ако на задача во `constants.ts` додадете `styleVariant: 'notebook'`, таа ќе се прикаже со позадина на квадратчиња (како Задача 9).
*   **Speech Bubble:** Можете да додадете `speechBubble: { speaker: 'Name', gender: 'male', text: '...' }` за да прикажете лик што зборува.
